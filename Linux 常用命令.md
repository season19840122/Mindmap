# Linux
[TOC]

### 包管理软件

```linux
# Windows 下：Chocolatey 
choco

# MacOS 下：Homebrew 
brew

# Linux 下：Yum、apt-get、rpm
# Redhat、Centos、Fedora 包管理工具，不仅可以安装本地包还可以安装线上包
Yum
# Ubuntu、Debian的包管理工具
apt-get
# rpm 文件关联性太大，而且只能安装本地包,一般不建议使用
rpm
```

### 常用命令

```linux
# 查看本地 IP:
ifconfig

# 编辑文件
vim 文件名

# 查看局域网 IP
arp -a

# CentOS 文字界面网络设定
ntmui
```
